<amature 끼리 연결성 부여하기>

1. object mode - amature 선택 - edit mode - data - viewport display - display as: B-bone (뼈 사이즈 조절 가능한 옵션으로 amature 모양 변경)

2. then, 창에서 - amature - transform - B.bone scale로 전체 B.bone사이즈 줄이고
3. 발바닥 B.bone만 duplicate후 두껍게 만들고
   해당 B.bone만 active element로 pivot point 바꿔서 사이즈 상당히 길이를 줄임 
	(이 놈이 추후 IK의 손잡이임)

<IK, inverse kinematics로 다리 움직임 구현>

1. 작아진 네모, 손잡이 B.bone 클릭 후, 윗 다리 뼈, 추가 클릭 후,
	ctrl + shift + c - IK, inverse kinematics 추가

2. g로 움직임 확인 (root가 연결되어 있음. 고쳐야 함 = bone constraint에서 chain length 값을 조절하면 됨.)

3. 타겟 본인 윗 다리 뼈 클릭 후, 디테일 창에서, bone constraint에서 chain length 값을 적절하게 조절.
